# Tarea-3-ADTD
Repositorio que almacena el código fuente de la tarea #3 de Análisis de Datos

Este proyecto es una aplicación web interactiva desarrollada en Python utilizando el framework Dash y el microservidor Flask. En el, se muestra información recopilada y procesada del dataset "Forest Fires".

Para ejecutar este dashboard de forma local, deberá tener instaladas las siguientes librerías: numpy, scipy, scikit-learn, dash, flask y plotly.
